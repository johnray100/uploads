Ne használjon Apache Tomcat csomagokat és API-kat

Ez a szabály a Java fájlokat elemezve észleli a következő Apache Tomcat szabadalomvédett csomagok használatát:

  • org.apache.tomcat
  • org.apache.catalina
  • org.apache.coyote
  • org.apache.el
  • org.apache.jasper
  • org.apache.jk
  • org.apache.juli
  • org.apache.naming

Amikor alkalmazása ezen csomagokban található osztályokat használ, akkor a kódot módosítani kell, úgy, hogy az Java fejlesztőkészlet (JDK) által biztosított, WebSphere Application Server által biztosított vagy nyílt forrású alkalmazásprogramozási felületeket használjon.

Értékelje ki a JBoss rendszerre jellemző alkalmazásprogramozási felületet használó kódot, majd módosítsa úgy, hogy az egy nem szabadalomvédett vagy WebSphere Application Server alkalmazásprogramozási felületet használjon.
