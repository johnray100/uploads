Java EE telepítésleírók használata a kontextus életciklus figyelők megadásához

Ez a szabály az összes Context <Listener> Elemek a META-INF/context.xml fájlban.

A Apache Tomcat, Java objektumok, amelyeknek a kontextus elindításakor vagy leállításakor tudniuk kell, hogy egy Listener Elem a környezetben. A megadott osztálynév Listener Elem megvalósítja a org.apache.catalina.LifecycleListener Interfész. Ez az osztály kap értesítést a megfelelő életciklus események előfordulásáról. Megjegyzés: Ne használja az Apache Tomcat csomagokat és API-k szabály jelzők előfordulását org.apache.catalina.LifecycleListener .

Az analóg Java EE funkció elérhető javax.servlet.ServletContextListener és ServletContextEvent és a megfelelő Listener Bejegyzések a web.xml fájlban.

Tomcat lehetővé teszi Context <Listener> Az alábbi helyeken meghatározott elemek:

  • A META-INF/context.xml fájlban
  • Az alkalmazás XML konfigurációs fájljában a Tomcat conf könyvtárában
  • A <Host> Elem a conf/server.xml elemben

Az áttérési eszköz csak jelzőket Listener Elemek a META-INF/context.xml fájlban. Listener A többi helyen megadott információk átmásolhatók a META-INF/context.xml fájlba annak feldolgozásához.

Az ajánlott művelet a Java fájlok konvertálása org.apache.catalina.LifecycleListener A használat javax.servlet.ServletContextListener és adja hozzá a megfelelő figyelőbejegyzéseket az alkalmazás web.xml fájljához.
