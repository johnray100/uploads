Megosztási hatókör beállítás Erőforrás-hivatkozásokon

Ez a szabály a következőket jelöli meg: resource-ref A WEB-INF/web.xml fájlban található elemek, amelyek hiányoznak res-sharing-scope Elem. Apache Tomcat kiszolgáló alapértelmezett erőforrás-megosztási hatóköre Shareable . Az erőforrás-megosztási hatókör a Java EEválasztható attribútumként van meghatározva. A forráselemzőben rendelkezésre áll egy gyorsjavítás ehhez a szabályhoz. A gyorsjavítás használatával ugyanazt a megosztható hatókört fogja kapni a hagyományos WebSphere és Liberty kiszolgálón, mint amivel a Tomcat kiszolgálón rendelkezett.

A gyorsjavítás hozzáadja a res-sharing-scope Elem, ha még nincs beállítva és meghatározza a kapcsolatot Shareable .

Az alábbi példában olyan erőforrás hivatkozás látható, amely megjelölésre kerülne:
< span class="Code"> < resource-ref>
< átívelő osztály= "indent4"> < /span> < leíró >
Példa adatbázis</description>
<res-ref-name>jdbc/exampleDB</res-ref-name>
<res-type>javax.sql.DataSource</res-type>
<res-auth>Tároló</res-auth>
</resource-ref>


A gyorsjavítás hozzáadja a res-sharing-scope elemet:
< span class="Code"> < resource-ref>
< átívelő osztály= "indent4"> < /span> < leíró >
Példa adatbázis</description>
<res-ref-name>jdbc/exampleDB</res-ref-name>
<res-type>javax.sql.DataSource</res-type>
<res-auth>Tároló</res-auth>
<res-sharing-scope>Shareable</res-sharing-scope>
</resource-ref>

A kapcsolatok WebSphere Application Server környezetben történő megosztásával kapcsolatos további információkat a következő helyen talál: Megoszthatatlan és megosztható kapcsolatok.
