Ne használjon Context Valve komponenst

Ez a szabály az összes Context <Valve> Elemek a META-INF/context.xml fájlban.

A funkcionalitás Valve Az összetevő a Java EE alkalmazásban érhető el Java szervlet szűrők használatával. Az ajánlott művelet a Java szervlet szűrők használata az Apache Tomcat helyett Valve Komponens.

A Tomcat Valve komponensével kapcsolatos további információkért lásd:

  • Apache Tomcat konfigurációs leírás - A Valve komponens.

A Java szervletszűrőkkel kapcsolatos további információkért lásd:

  • Szűrőkkel kapcsolatos lényeges információk.
