Не следует использовать метод WsServerContainer doUpgrade Apache Tomcat

В Liberty V8.5.5.7 и WebSphere Application Server traditional V9.0 добавлен метод WebSocket com.ibm.websphere.wsoc.WsWsocServerContainer doUpgrade, который позволяет сервлету или фильтру запрашивать возможность обновления текущего запроса HTTP до сеанса WebSocket. В Apache Tomcat аналогичная функциональность достигается с помощью метода org.apache.tomcat.websocket.server.WsServerContainer doUpgrade.

Это правило отмечает метод Tomcat WsServerContainer doUpgrade, чтобы его можно было заменить на метод WsWsocServerContainer doUpgrade.

Метод WebSphere Application Server определен следующим образом: 


com.ibm.websphere.wsoc.WsWsocServerContainer

Недействительной doUpgrade(HttpServletRequest request, 
                      HttpServletResponse response,
                      ServerEndpointConfig sec, 
                      java.util.Map<java.lang.String,
                      java.lang.String> pathParams) 
                        throws ServletException, java.io.IOException

}
