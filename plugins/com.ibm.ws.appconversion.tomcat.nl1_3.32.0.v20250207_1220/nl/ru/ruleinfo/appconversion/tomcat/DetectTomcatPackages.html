Не следует использовать API и пакеты Apache Tomcat

Это правило ищет в файлах Java случаи использования следующих патентованных пакетов Apache Tomcat:

  • org.apache.tomcat
  • org.apache.catalina
  • org.apache.coyote
  • org.apache.el
  • org.apache.jasper
  • org.apache.jk
  • org.apache.juli
  • org.apache.naming

Если приложение использует классы из этих пакетов, необходимо изменить исходный код и использовать API, предоставляемые JDK или WebSphere Application Server, либо использовать API с открытым исходным кодом.

Оцените исходный код, использующий API Tomcat, и измените его для использования непатентованных API или API WebSphere Application Server.

Дополнительная информация:

  • API Apache Tomcat 8.0 - документация Java.
  • API Apache Tomcat 9.0 - документация Java.
