Nepoužívat balíky Apache Tomcat a rozhraní API

Toto pravidlo prochází a hledá soubory Java za účelem zjištění použití následujících proprietárních balíků Apache Tomcat:

  • org.apache.tomcat
  • org.apache.catalina
  • org.apache.coyote
  • org.apache.el
  • org.apache.jasper
  • org.apache.jk
  • org.apache.juli
  • org.apache.naming

Když vaše aplikace používá v těchto balících třídy, musí se kód změnit tak, aby používal rozhraní API poskytovaná sadou JDK nebo serverem WebSphere Application Server nebo rozhraní API typu open source.

Vyhodnoťte kód pomocí rozhraní API specifického pro Tomcat a změňte jej tak, aby používal neproprietární rozhraní API nebo rozhraní API serveru WebSphere Application Server.
