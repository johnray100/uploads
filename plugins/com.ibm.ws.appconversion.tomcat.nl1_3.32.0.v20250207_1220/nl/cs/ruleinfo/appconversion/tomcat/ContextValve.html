Nepoužívat komponentu Context Valve

Toto pravidlo značí vše Context <Valve> prvky v souboru META-INF/context.xml .

Funkce zařízení Valve Komponenty lze dosáhnout v prostředí Java EE pomocí filtrů servletu Java . Doporučenou akcí je použití filtrů servletu Java místo Apache Tomcat Valve komponenty.

Další informace o komponentě Valve na serveru Tomcat viz:
