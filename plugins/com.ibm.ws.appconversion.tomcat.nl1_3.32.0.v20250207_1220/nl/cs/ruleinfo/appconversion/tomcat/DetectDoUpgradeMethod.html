Nepoužívat metodu Apache Tomcat WsServerContainer doUpgrade

Liberty V8.5.5.7 a WebSphere Application Server V9.0 Traditional zavádí metodu com.ibm.websphere.wsoc.WsWsocServerContainer doUpgrade WebSocket, která umožňuje servletu nebo filtru požádat, aby byl aktuální požadavek HTTP upgradován na relaci WebSocket. Apache Tomcat poskytuje podobnou schopnost prostřednictvím metody org.apache.tomcat.websocket.server.WsServerContainer doUpgrade.

Toto pravidlo značí metodu Tomcat WsServerContainer doUpgrade, takže ji můžete migrovat do metody WsWsocServerContainer doUpgrade.

Metoda serveru WebSphere Application Server je definována následujícím způsobem: 


com.ibm.websphere.wsoc.WsWsocServerContainer

public void doUpgrade(HttpServletRequest request, HttpServletResponse response, ServerEndpointConfig sec, java.util.Map<java.lang.String, java.lang.String> pathParams) vyvolá ServletException, java.io.IOException }
