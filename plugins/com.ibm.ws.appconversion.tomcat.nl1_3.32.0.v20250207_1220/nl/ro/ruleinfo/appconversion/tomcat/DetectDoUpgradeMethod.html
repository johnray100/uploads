Nu utilizaţi metoda Apache Tomcat WsServerContainer doUpgrade

Liberty V8.5.5.7 şi WebSphere Application Server V9.0 tradiţional introduc metoda com.ibm.websphere.wsoc.WsWsocServerContainer doUpgrade WebSocket, care permite unui servlet sau filtru să ceară ca cererea HTTP curentă să fie modernizată la o sesiune WebSocket. Apache Tomcat oferă capacităţi similare prin metoda org.apache.tomcat.websocket.server.WsServerContainer doUpgrade.

Această regulă semnalează metoda Tomcat WsServerContainer doUpgrade astfel încât să o puteţi migra la metoda WsWsocServerContainer doUpgrade.

Metoda WebSphere Application Server este definită după cum urmează: 


com.ibm.websphere.wsoc.WsWsocServerContainer

vid public doUpgrade(HttpServletRequest request, 
                      HttpServletResponse response,
                      ServerEndpointConfig sec, 
                      java.util.Map<java.lang.String,
                      java.lang.String> pathParams) 
                        aruncă ServletException, java.io.IOException

}
