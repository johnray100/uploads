Nu utilizaţi componenta Context Valve

Această regulă steaguri toate Context <Valve> elemente în META-INF/context.xml .

Funcționalitatea sistemului de Valve Componenta poate fi realizată în Java EE filtre de servomotor Java Java Acțiunea recomandată este de a utiliza filtre de tip " Java în loc de Apache Tomcat Valve component.
