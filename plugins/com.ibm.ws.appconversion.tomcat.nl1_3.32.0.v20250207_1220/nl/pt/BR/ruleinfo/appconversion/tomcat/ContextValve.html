Não usar o componente de válvula de contexto

Esta regra sinaliza todos Context <Valve> no arquivo META-INF/context.xml .

A funcionalidade do Valve pode ser obtido no Java EE usando filtros de servlet Java . A ação recomendada é usar filtros de servlet Java em vez do Apache Tomcat Valve componente.

Para obter informações adicionais sobre o componente da válvula no Tomcat, veja:

  • Referência de configuração do Apache Tomcat - O componente Valve.

Para obter informações adicionais sobre os filtros do servlet Java:

  • Os elementos essenciais de filtros.
