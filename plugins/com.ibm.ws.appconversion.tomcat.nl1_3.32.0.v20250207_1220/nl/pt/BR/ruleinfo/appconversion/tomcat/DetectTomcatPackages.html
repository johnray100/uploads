Não Usar os Pacotes Apache Tomcat e APIs

Esta regra varre os arquivos Java para detectar o uso dos pacotes proprietários Apache Tomcat a seguir:

  • org.apache.tomcat
  • org.apache.catalina
  • org.apache.coyote
  • org.apache.el
  • org.apache.jasper
  • org.apache.jk
  • org.apache.juli
  • org.apache.naming

Quando o aplicativo utilizar classes nesses pacotes, o código deverá ser alterado para utilizar APIs fornecidas pelo Java Development Kit (JDK), APIs fornecidas pelo WebSphere Application Server ou APIs de software livre.

Avalie o código usando a API específica do Tomcat e altere-o para usar uma API não proprietária ou uma API do WebSphere Application Server.

Para obter informações adicionais, consulte:

  • Documentação Java da API do Apache Tomcat 8.0 .
  • Documentação Java do Apache Tomcat 9.0 .
