Nie używaj komponentu Context Valve

Ta reguła powoduje oznaczenie wszystkich Context <Valve> elementy w pliku META-INF/context.xml .

Funkcjonalność Valve Komponent można uzyskać w środowisku Java EE przy użyciu filtrów serwletu Java . Zalecanym działaniem jest użycie filtrów serwletu Java zamiast serwera Apache Tomcat . Valve .
