Kontextkomponente valve nicht verwenden

Diese Regel markiert alle Context <Valve> Elemente in der Datei META-INF/context.xml .

Die Funktionalität der Valve Komponente kann in Java EE mit Java -Servletfiltern erreicht werden. Es wird empfohlen, Java -Servletfilter anstelle von Apache Tomcat zu verwenden. Valve Komponente.
