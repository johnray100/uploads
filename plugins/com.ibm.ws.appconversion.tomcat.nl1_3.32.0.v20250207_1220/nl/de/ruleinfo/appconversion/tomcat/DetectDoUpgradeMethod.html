Methode Apache Tomcat WsServerContainer doUpgrade nicht verwenden

Liberty Version 8.5.5.7 und WebSphere Application Server Traditional Version 9.0 führen die WebSocket-Methode com.ibm.websphere.wsoc.WsWsocServerContainer doUpgrade ein, mit der ein Servlet oder ein Filter eine Aktualisierung der aktuellen HTTP-Anforderung auf eine WebSocket-Sitzung anfordern kann. Apache Tomcat stellt mit der Methode org.apache.tomcat.websocket.server.WsServerContainer doUpgrade eine vergleichbare Funktion bereit.

Diese Regel markiert die Tomcat-Methode WsServerContainer doUpgrade, damit Sie diese auf die Methode WsWsocServerContainer doUpgrade migrieren können.

Die WebSphere Application Server-Methode ist wie folgt definiert: 


com.ibm.websphere.wsoc.WsWsocServerContainer

public void doUpgrade(HttpServletRequest request, HttpServletResponse response, ServerEndpointConfig sec, java.util.Map<java.lang.String, java.lang.String> pathParams) throws ServletException, java.io.IOException }
