No utilizar BasicDataSource de Apache Tomcat

Esta regla detecta el uso del org.apache.tomcat.dbcp.dbcp.BasicDataSource en archivos Java . Esta clase implementa la interfaz de origen de datos estándar del sector javax.sql.DataSource . El origen de datos Apache Tomcat proporciona métodos de propiedad que la aplicación puede utilizar.

Si no está utilizando métodos de propiedad, puede cambiar el código para utilizar javax.sql.DataSource en lugar de convertir a la Versión de Tomcat .

Si utiliza los métodos de propietario y desea seguir utilizándolos en la aplicación, puede migrar el recurso para utilizar Apache Commons DataSource. Para utilizar Apache Commons, descargue el archivo jar de dbcp común y configure el recurso para utilizar este origen de datos. En el código Java , cambie las referencias de org.apache.tomcat.dbcp.dbcp.BasicDataSource por: org.apache.commons.dbcp.BasicDataSource .

Como alternativa, puede volver a configurar el recurso para utilizar las agrupaciones de conexiones de WebSphere y aumentar el rendimiento y reducir los problemas de generación de hebras y escalabilidad. Utilizando esta solución, están disponibles la información y las estadísticas de origen de datos para la aplicación utilizando el MBean de fábrica de conexiones.
