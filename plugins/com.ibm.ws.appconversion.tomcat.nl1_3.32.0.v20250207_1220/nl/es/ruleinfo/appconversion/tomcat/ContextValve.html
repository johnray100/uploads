No utilizar el componente Context Valve

Esta regla marca todo Context <Valve> en el archivo META-INF/context.xml .

La funcionalidad del Valve componente se puede conseguir en Java EE utilizando filtros de servlet Java . La acción recomendada es utilizar filtros de servlet Java en lugar de Apache Tomcat Valve componente.

Para obtener información adicional sobre el componente Valve en Tomcat, consulte:

  • Referencia de configuración de Apache Tomcat - El componente Valve.

Para obtener información adicional sobre los filtros de servlet Java:

  • Los conceptos básicos de filtros.
