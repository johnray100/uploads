Dans Apache Tomcat, les objets Java qui doivent savoir quand leur contexte est démarré ou arrêté sont déclarés en incluant un Listener dans le contexte. Le nom de classe spécifié dans le Listener implémente l'élément org.apache.catalina.LifecycleListener CredentialGenerator. La fonctionnalité Java EE analogue est disponible à l'aide de javax.servlet.ServletContextListener et ServletContextEvent et la valeur correspondante Listener dans le fichier web.xml .

Tomcat autorise Context <Listener> éléments à définir dans ces emplacements:

  • dans le fichier META-INF/context.xml,
  • dans le fichier XML d'application dans le répertoire conf de Tomcat,
  • :NONE. <Host> dans le fichier conf/server.xml
