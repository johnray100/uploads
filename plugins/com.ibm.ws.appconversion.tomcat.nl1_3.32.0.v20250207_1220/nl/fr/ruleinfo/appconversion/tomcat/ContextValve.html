Ne pas utiliser le composant Context Valve

Cette règle marque tout Context <Valve> dans le fichier META-INF/context.xml .

La fonctionnalité de la Valve peut être réalisée dans Java EE à l'aide de filtres de servlet Java . L'action recommandée consiste à utiliser des filtres de servlet Java à la place d' Apache Tomcat Valve composant.

Pour plus d'informations sur le composant Valve dans Tomcat, voir :

  • Apache Tomcat Configuration Reference - The Valve Component.

Pour plus d'informations sur les filtres de servlet Java, voir :

  • The Essentials of Filters.
