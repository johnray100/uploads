Non utilizzare il metodo Apache Tomcat WsServerContainer doUpgrade

Liberty V8.5.5.7 WebSphere Application Server traditional V9.0 introducono il metodo com.ibm.websphere.wsoc.WsWsocServerContainer doUpgrade WebSocket, che abilita un servlet o filtro a richiedere che la richiesta HTTP corrente venga aggiornata per una sessione WebSocket. Apache Tomcat fornisce una funzionalit√† simile attraverso il metodo org.apache.tomcat.websocket.server.WsServerContainer doUpgrade.

Questa regola contrassegna il metodo WsServerContainer doUpgrade in modo che sia possibile migrarlo nel metodo WsWsocServerContainer doUpgrade.

Il metodo WebSphere Application Server viene definito come riportato di seguito:
