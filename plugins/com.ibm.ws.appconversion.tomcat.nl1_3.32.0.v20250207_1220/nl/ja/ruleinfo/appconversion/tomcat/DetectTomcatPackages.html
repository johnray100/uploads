Apache Tomcat のパッケージおよび API を使用しない

この規則は、Java ファイルをスキャンして、以下の Apache Tomcat 専有パッケージの使用を検出します。

  • org.apache.tomcat
  • org.apache.catalina
  • org.apache.coyote
  • org.apache.el
  • org.apache.jasper
  • org.apache.jk
  • org.apache.juli
  • org.apache.naming

アプリケーションでこれらのパッケージにあるクラスを使用する場合は、Java Development Kit (JDK) 提供の API、WebSphere Application Server 提供の API、またはオープン・ソース API を使用するようにコードを変更する必要があります。

Tomcat 固有の API を使用するコードを評価し、専有以外の API または WebSphere Application Server API を使用するように変更します。

  • Apache Tomcat 8.0 API に関する Java 資料。
  • Apache Tomcat 9.0 API に関する Java 資料。
