Apache Tomcat WsServerContainer doUpgrade メソッドを使用しない

Liberty V8.5.5.7 および WebSphere Application Server traditional V9.0 で com.ibm.websphere.wsoc.WsWsocServerContainer doUpgrade WebSocket メソッドが導入され、 これによって、現行 HTTP 要求が WebSocket セッションにアップグレードされることをサーブレットまたはフィルターが要求することが可能になりました。 Apache Tomcat は、これと同様の機能を org.apache.tomcat.websocket.server.WsServerContainer doUpgrade メソッドで提供します。

この規則により、WsWsocServerContainer doUpgrade メソッドにマイグレーションできるように、Tomcat WsServerContainer doUpgrade メソッドにフラグを立てます。

WebSphere Application Server メソッドは次のように定義されます。 


com.ibm.websphere.wsoc.WsWsocServerContainer

public void doUpgrade(HttpServletRequest request request, HttpServletResponse response, ServerEndpointConfig sec, java.util.Map<java.lang.String, java.lang.String> pathParams) throws ServletException, java.io.IOException }
