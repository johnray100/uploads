不要使用 Apache Tomcat 套件和 API

此規則會掃描 Java 檔案，以偵測是否使用下列的 Apache Tomcat 專有套件：

  • org.apache.tomcat
  • org.apache.catalina
  • org.apache.coyote
  • org.apache.el
  • org.apache.jasper
  • org.apache.jk
  • org.apache.juli
  • org.apache.naming

如果應用程式使用這些套件中的類別，則必須將程式碼變更為使用 Java 開發套件 (JDK) 提供的 API、WebSphere Application Server 提供的 API 或開放程式碼 API。

使用 Tomcat 專用 API 來評估程式碼，並將程式碼變更為使用非專有的 API 或 WebSphere Application Server API。

如需相關資訊，請參閱：

  • Apache Tomcat 8.0 API Java 說明文件。
  • Apache Tomcat 9.0 API Java 說明文件。
