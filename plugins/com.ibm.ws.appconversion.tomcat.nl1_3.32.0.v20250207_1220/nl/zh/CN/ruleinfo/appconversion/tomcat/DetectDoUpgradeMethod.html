请勿使用 Apache Tomcat WsServerContainer doUpgrade 方法

Liberty V8.5.5.7 和传统 WebSphere Application Server V9.0 中引入了 com.ibm.websphere.wsoc.WsWsocServerContainer doUpgrade WebSocket 方法，该方法使 Servlet 或过滤器能够请求将当前 HTTP 请求升级到 WebSocket 会话。 Apache Tomcat 通过 org.apache.tomcat.websocket.server.WsServerContainer doUpgrade 方法提供类似的功能。

该规则标记 Tomcat WsServerContainer doUpgrade 方法，以便您可以将其迁移到 WsWsocServerContainer doUpgrade 方法。

WebSphere Application Server 方法定义如下： 


com.ibm.websphere.wsoc.WsWsocServerContainer

公共空 doUpgrade(HttpServletRequest request, 
                      HttpServletResponse response,
                      ServerEndpointConfig sec, 
                      java.util.Map<java.lang.String,
                      java.lang.String> pathParams) 
                        抛掷 ServletException, java.io.IOException

}
