请勿使用 Apache Tomcat 包和 API

此规则会扫描 Java 文件以检测是否使用了下列 Apache Tomcat 专有包：

  • org.apache.tomcat
  • org.apache.catalina
  • org.apache.coyote
  • org.apache.el
  • org.apache.jasper
  • org.apache.jk
  • org.apache.juli
  • org.apache.naming

如果应用程序使用这些包中的类，那么必须更改代码以使用 Java Development Kit (JDK) 提供的 API、WebSphere Application Server 提供的 API 或者开放式源代码 API。

评估使用特定于 Tomcat 的 API 的代码，并更改它以使用非专有 API 或使用 WebSphere Application Server API。

有关更多信息，请参阅：

  • Apache Tomcat 8.0 API Java 文档。
  • Apache Tomcat 9.0 API Java 文档。
